# To-Do List

A simple To-Do List web application built using HTML, CSS, and JavaScript. Users can add, delete, and manage their tasks.

---

## **Features**
- Add new tasks.
- Delete tasks.
- Save tasks to local storage (persist on page refresh).
- Clean and responsive design.
